Manual de uso

Instrucciones para la modificación de registros

Este formulario le permite modificar los datos de un libro o material en el catálogo de la biblioteca.

Campos obligatorios
  • Título: Nombre completo del libro o material.
  • Clasificación / CDU: Código según el sistema de Clasificación Decimal Universal.
  • Siglas de autor / título: Primeras letras del apellido del autor (normalmente 3 letras).
  • Sede: Ubicación física del material.
  • Disponibilidad: Estado actual para préstamo.
Campos opcionales
  • Autor: Nombre del autor en formato "Apellido, Nombre".
  • Editorial: Nombre de la editorial que publicó el material.
  • Etiquetas: Palabras clave separadas por comas para facilitar la búsqueda.
  • Descripción: Resumen o sinopsis del contenido.
  • Observaciones: Comentarios adicionales sobre el estado físico u otros aspectos.
  • Imagen: Portada o imagen representativa del material.
Funciones especiales
  • Generar resumen: Crea automáticamente un resumen a partir de la descripción completa.
  • Cargar imagen: Permite añadir una imagen desde su dispositivo o mediante URL.
