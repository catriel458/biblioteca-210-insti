MANUAL DE ALTA DE MATERIAL

Introducción

Bienvenido al sistema de Alta de Material. Esta pantalla te permite registrar nuevos materiales en el sistema bibliotecario, ya sean libros, mapas, dispositivos multimedia, proyectores o notebooks. A continuación, te explicamos paso a paso cómo utilizar correctamente todas las funciones.

Pasos para dar de alta un material

1. Selección del tipo de material

En la parte superior del formulario encontrarás un desplegable donde debes seleccionar el tipo de material que deseas registrar:

  • Libro: Para registrar libros, textos, revistas y material impreso.
  • Mapa: Para registrar material cartográfico.
  • Multimedia: Para registrar CDs, DVDs, u otro contenido audiovisual.
  • Proyector: Para registrar equipos de proyección.
  • Notebook: Para registrar computadoras portátiles.

2. Cantidad de ejemplares

Indica cuántos ejemplares del mismo material deseas registrar simultáneamente. Si ingresas más de uno, se crearán filas adicionales en el formulario para que puedas asignar números de inventario diferentes a cada ejemplar.

3. Completar el formulario específico

Después de seleccionar el tipo de material, aparecerá automáticamente un formulario con los campos específicos para ese tipo. Completa todos los campos requeridos (marcados con *).

4. Carga masiva

Si necesitas registrar muchos materiales del mismo tipo, puedes utilizar la función de carga masiva:

  • Haz clic en el botón de carga (flecha hacia arriba) ubicado en la esquina superior derecha.
  • Selecciona el archivo Excel o CSV con los datos de los materiales a importar.
  • El sistema procesará el archivo y cargará los datos automáticamente.

Nota: Asegúrate de que tu archivo de carga masiva tenga el formato correcto. Consulta con el administrador del sistema si tienes dudas sobre la estructura del archivo.

Campos específicos por tipo de material

Para Libros:

  • Título: Nombre completo del libro.
  • Autor: Nombre del autor o autores.
  • Editorial: Nombre de la editorial.
  • Año de publicación: Año en que se publicó el libro.
  • ISBN: Código internacional normalizado del libro.
  • Categoría: Área temática a la que pertenece.

Para Mapas:

  • Título: Nombre del mapa.
  • Región/Área: Zona geográfica que representa.
  • Escala: Relación entre la dimensión real y la representada.
  • Año de edición: Año en que se editó el mapa.

Para Multimedia:

  • Título: Nombre del contenido multimedia.
  • Tipo: CD, DVD, Blu-ray, etc.
  • Duración: Tiempo de reproducción.
  • Formato: Formato del archivo o soporte.

Para Proyectores y Notebooks:

  • Marca: Fabricante del equipo.
  • Modelo: Modelo específico.
  • Número de serie: Identificador único del dispositivo.
  • Características: Especificaciones técnicas relevantes.

Botones de acción

Al final del formulario encontrarás dos botones:

  • Cancelar: Borra todos los datos ingresados y regresa a la pantalla anterior sin guardar cambios.
  • Guardar: Guarda toda la información ingresada y registra el material en el sistema.

Consejos útiles

  • Antes de guardar, verifica que todos los datos estén correctamente ingresados.
  • Si necesitas registrar varios ejemplares del mismo material pero con alguna diferencia entre ellos, usa la opción de cantidad de ejemplares e ingresa los datos específicos en cada fila.
  • El sistema asignará automáticamente un número de inventario, pero puedes modificarlo si es necesario.
  • Si tienes dudas sobre algún campo específico, coloca el cursor sobre la etiqueta del campo para ver información adicional.
  • Para cualquier consulta adicional o reporte de problemas, comunícate con el soporte técnico.
